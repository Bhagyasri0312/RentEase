### To Customize:

**Change Colors:**

```jsx
// Update class names in Home.jsx
className={`bg-gradient-to-br ${category.color}`}
// Change color values: blue-600 → red-600, green-600, etc.
```

**Add Products:**

```jsx
// Add to featuredProducts array
{
  id: '9',
  name: 'New Product',
  price: 2999,
  originalPrice: 3999,
  // ... other properties
}
```

**Modify Categories:**

```jsx
// Update categories array
{
  name: 'New Category',
  icon: IconComponent,
  image: 'image-url',
  // ... other properties
}
```
