### Change Primary Color

```jsx
// Find in Home.jsx and replace:
blue-600 → red-600
blue-400 → red-400
```

### Add New Category

```jsx
// Add to categories array:
{
  name: 'New Item',
  icon: IconName,
  image: 'url',
  color: 'from-color-400 to-color-600',
  count: '100+ items',
}
```
